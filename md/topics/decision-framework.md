# Decision framework

Required before every new investment.

## Questions

1. **Why this company?** — business quality, edge, fit in portfolio
2. **Why now?** — catalyst, price, timing vs macro
3. **Why not later?** — what changes if you wait
4. **Biggest risks** — what kills the thesis
5. **Worst-case scenario** — capital at risk
6. **Expected upside** — target, R/R
7. **Maximum acceptable loss** — stop, position size
8. **Exit conditions** — technical, fundamental, time stop

## Experiment layer (H001–H030)

Additionally enforced by app:

- ID in H001–H030 range
- Cycle loss limit -300 USD
- Stop required on every trade
- No new trades when limit breached

See `md/rules/experiment-cycle.md`.

## Document

Record answers in:

- Trade note body (experiment)
- `companies/TICKER/thesis.md` (long-term)
- `journal/` (decision log)
