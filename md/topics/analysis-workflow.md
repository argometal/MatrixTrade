# Analysis workflow

Default process before any trade.

## Timeframe sequence

```
3 Month chart
    ↓
1 Month chart
    ↓
1 Week chart
    ↓
1 Day chart (entry timing only)
    ↓
Fundamental analysis
    ↓
Valuation
    ↓
Decision
```

## Rule

> No trade unless higher timeframes support the thesis.

Daily chart is for entry timing — not for overriding higher-timeframe structure.

## Where to document

| Step | Location |
|------|----------|
| Chart notes | Trade body · `companies/TICKER/charts/` |
| Fundamentals | `companies/TICKER/thesis.md` |
| Valuation | `companies/TICKER/valuation.md` |
| Final decision | `journal/` + trade note |
