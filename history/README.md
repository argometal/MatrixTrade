# History

Append-only audit trail. Never delete — add new entries.

## Use for

- Snapshot exports sent to ChatGPT (dated copies)
- Corrections or clarifications (never overwrite closed trades)
- Cycle summaries, portfolio state at a point in time

## Naming

```
history/2026-03-29-cycle-snapshot.txt
history/2026-03-29-H001-correction-note.md
```
