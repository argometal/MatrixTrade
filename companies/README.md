# Companies

One folder per ticker. All history for a company stays together.

## Structure

```
companies/
  GOOG/
    thesis.md       Business, moat, risks, valuation, bull/bear
    valuation.md    Fair value, expected return
    trades.md       Index: H007, H012 → links to trade files
    earnings/       Quarterly notes
    charts/         Snapshots
    news/           Articles and summaries
```

## Why

If you make 40 trades in Google over five years, everything remains under `companies/GOOG/`.
