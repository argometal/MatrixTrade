Dirección requerida:

- Close trade       → trade-close (confirmExternalClose: true si pending pero ya ejecutado fuera)
- Analyze trade     → analysis (thesis, psychology, lessons, notes)
- Review trade      → trade-review (post-cierre: quality 1-5, mistakes, lesson)
- Playbook / style  → playbook-create / playbook-update / trade-update.playbookId

- UI muestra trade-proposal, trade-update al usuario (demasiado tecnico)
